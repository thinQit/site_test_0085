# Convertly Landing

A clean, high-conversion landing page scaffold built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **shadcn/ui**.

## Stack

- Next.js 14.2.21 (App Router)
- React 18.3.1
- TypeScript 5.6.3
- Tailwind CSS 3.4.17
- shadcn/ui
- Framer Motion 11.15.0
